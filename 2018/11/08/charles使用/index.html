电脑安装证书

之后会弹出钥匙串，如果不弹出，请自行打开钥匙串.
系统默认是不信任 Charles 的证书的，此时对证书右键，在弹出的下拉菜单中选择『显示简介』，点击使用此证书时，把使用系统默认改为始终信任

手机设备安装证书

设置服务器代理
ip 为你本机的 ip，端口 8888
然后打开手机的浏览器，输入
charlesproxy.com/getssl

使用 SSL 代理

操作之后依旧无法抓取 https 包，显示 unknown？
